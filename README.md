# Chess

Chess game for a school project.
## Technologies
* Java 11
* AWT/Swing
* Gradle 7.4

## Authors
* Oleksandr Danchenko 
* Parsa Hemmati
* Samuel Browne

## How to run
To run as gradle task:
```bash
./gradlew run
```
To generate .jar: 
```bash
./gradlew jar
```
To execute .jar file:
```bash
java -jar Chess-1.0.jar
```
